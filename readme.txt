== Description ==

Memory Viewer is a WordPress plugin that allows you to view WordPress' memory utilization .

 It is very simple. All you need to do is:
1. Enable the plugin
2. Visit your WordPress site
3. View the page source and look at the bottom of the page.
4. When you are done troubleshooting the plugin, simply disable it
